## V2 amendment: reliability-calibrated fixed-rank connection

上一版把 LID 估计直接映射为每个 cell 的 fiber rank。这会把 rank change、LID 饱和与 holonomy 混在一起。V2 已修改为：

1. 主 connection 使用预注册固定秩 (q\in\{2,4,6\})；默认 (q=6)。
2. LID 与 numerical rank 只生成 `fiber_rank` 和 `rank_singularity` field，不决定主 connection rank。
3. 主曲率改为 Wilson loop eigenangle curvature：

\[
W_p=P_{\ell t}^{\top}P_{t\ell},
\qquad
\kappa_p^W=
\frac{1}{\pi}
\sqrt{\frac{1}{q}\sum_r\theta_r^2}.
\]

4. 每个 plaquette 同时保存四条 edge 的最差 Procrustes residual：

\[
\varepsilon_p=\max_{e\in\partial p}\varepsilon_e.
\]

5. 主分析使用开发集冻结阈值后的 `plaquette_reliable_wilson`；raw curvature、residual 与 reliable coverage 必须共同报告。
6. `fiber_rank_mode=lid` 仅作为历史 ablation，不进入主表。
7. `phase_shuffle` 与 `reference_id_shuffle` 通过独立 artifact 运行，禁止与主 field 混存或调参。

专用 evaluator 必须输出：

- first-error aligned raw/adjusted event curve；
- problem fixed effects + `rel_pos` + `log1p(step_len)` 调整；
- conditioning on LID/rank/rewiring/transport residual 后的 Wilson event；
- same-problem wrong-correct difference 与 pair-micro AUROC；
- Wilson/transport Spearman correlation 与 reliable fraction；
- per-layer event map，而不是只输出一个 max scalar。

V2 go/no-go 增加以下硬门：

- 固定秩 (q=2,4,6) 的 event direction 一致；
- reliability-gated curve 在 (	au=0) 形成局部峰，clustered CI 不跨 0；
- conditional Wilson local contrast 的 CI 不跨 0；
- Wilson/transport residual 的绝对 Spearman 小于 0.5；
- same-problem delta 的 clustered CI 不跨 0；
- phase/ID null 不保留同等强度的 first-error 局部带。

## 0. 决策摘要

本计划只检验一个统一几何对象，不训练锚点、hazard、干预器，也不把 LID、kNN 和 tangent 分数加权成 detector。

核心问题是：推理链在 \((t,\ell)\) 网格上的局部 tangent bundle 是否近似可积；错误附近是否出现稳定的 rank/front、邻域分叉与 plaquette holonomy 集中带。

当前状态：

- CPU schema、group split、label invariance、identical-layer、gauge invariance 已通过；
- exact artifact 可直接 token replay；
- sv_vec_mean 与 [step,layer,hidden] 两类输入已接通；
- 尚未完成真实 7B/8B 全层 GPU smoke，所以所有方法结论仍为 proposal，不是 empirical result。

## 1. 两个主张

### Claim 1：局部几何分叉

在按 problem_id 分组的 held-out 链上，first-error 附近出现局部而非单调位置效应的几何分叉：

\[
\Delta d_{i,t,\ell},
\qquad
\Omega^t_{i,t,\ell},
\qquad
\Omega^\ell_{i,t,\ell}
\]

相对于正确同题链和错误前状态发生一致变化，并跨数据集、模型与超参方向稳定。

否证条件：效应在控制 rel_pos、step length 和 problem fixed effect 后消失；峰只在链尾出现；或不同 \(k\)/JL/phase 参数下方向不稳定。

### Claim 2：layer-time 非交换性

错误附近的两轴局部 transport 更不相容：

\[
\kappa_{i,t,\ell}
=
\frac{
\left\|
R^t_{i,t,\ell}R^\ell_{i,t-1,\ell}
-
R^\ell_{i,t,\ell}R^t_{i,t,\ell-1}
\right\|_F
}{
2\sqrt q
}.
\]

该效应不能被 depth-only rewiring、time-only trajectory motion、LID-only 或位置/长度解释。

否证条件：holonomy 对 Claim 1 没有额外结构证据；time/depth shuffle 后不下降；identical-layer 或可分离 synthetic field 仍产生同等级曲率；或有效结果只来自 rank-singularity 样本。

## 2. 五个核心块

### Block A：exact whole-layer state grid

主输入：

\[
Z_i\in\mathbb R^{T_i\times L\times D}.
\]

硬要求：

- exact prompt/response token replay；
- step range 与 hidden states 同一 token 轴；
- raw hidden state；
- arithmetic mean over exact step tokens；
- 连续层；
- 保存 model/tokenizer/template/revision/seed。

### Block B：phase-matched problem-grouped reference

外层 5-fold 按 problem_id；训练参考按 problem→sample 两级平衡。每条训练链在查询进度 \(\rho_{i,t}\) 上插值，只贡献一个 phase-matched 状态。标签不参与 reference、投影或 neighborhood。

### Block C：local chart

同一个 kNN chart 同时产生：

- LID \(\widehat d_{i,t,\ell}\)；
- reference-chain neighbor lineage；
- 局部 tangent basis。

不再在 \(LD\) 拼接空间计算单一 LID，也不使用每层独立随机投影。

### Block D：rank-aware local connection 与 holonomy

fiber rank：

\[
q_{i,t,\ell}
=
\min(q_{\max},\operatorname{round}(\widehat d),\operatorname{rank}_{num}).
\]

相邻 cell 用同一参考链身份拟合 paired Procrustes connection；四角 rank 不同单独报告 rank_singularity，holonomy 在共同最小 rank 上计算。

### Block E：结构验证

完整 field 是主产物。预注册标量 reduction 只服务统计检验，不做任意总分：

- first-error aligned \(\kappa\) peak/AUC；
- depth band location 与 bandwidth；
- rank-singularity rate；
- correct-vs-wrong within-problem paired difference；
- 作为附属 readout 的 AUROC/Top-1，不作为创新定义。

## 3. 数据和切分

### 3.1 开发集

1. synthetic separable/interaction field；
2. 64 problems × 4 samples 的 exact-generation GPU smoke；
3. 至少同时含正确与错误 sample 的 internal multisample 子集。

### 3.2 确证集

- same-problem multisample：每题目标 \(K\ge 8\)，主测 response geometry；
- ProcessBench GSM8K/MATH/OlympiadBench：主测 first-error alignment；
- 至少一个不同模型家族作迁移。

若现有 artifact 只有 8 个稀疏层或 step_exp pooling，只进入 legacy ablation，不进入确证主表。

### 3.3 切分

- outer GroupKFold：5 folds，group=problem_id；
- 超参只在 outer-train 内固定或 nested group CV；
- 同一个 outer-test field 不能用于挑层、挑 \(k\) 再报告同 cohort；
- bootstrap 以 problem 为 cluster，建议 2000 次；
- 记录每 fold reference chains、coverage、缺失原因。

## 4. 三类基线

### Family 1：nuisance 与静态几何

- relative position、step length、chain length；
- hidden norm、step transition norm；
- per-layer LID-only、rewiring-only。

### Family 2：一维几何演化

- depth-only kNN overlap；
- time-only trajectory speed/tube；
- 现有 representation_geometry 与 spectral_chain_dynamics 修复 problem split 后的结果。

### Family 3：学习式上界

- linear/logistic probe；
- latent VAE；
- supervised latent separatrix。

Family 3 只能作为性能上界，不能反向定义几何机制。

## 6. 关键消融

仅围绕同一几何对象：

1. phase-matched vs pooled-all-time reference；
2. shared JL vs exact distance；独立逐层 JL 只作失败负控；
3. LID-adaptive rank vs fixed rank；
4. local paired connection vs global layer Procrustes；
5. holonomy vs depth-only/time-only observables；
6. normalized-step phase vs token-progress/arclength phase；
7. mean pooling vs step-exp 仅作 pooling sensitivity。

不进行 anchor、entropy、hazard、HMM、intervention 组合消融。

## 7. 统计协议

### 7.1 first-error event study

令 \(g_i\) 为首错步，窗口 \(\tau=t-g_i\)。对每个预注册 layer band 或全层摘要拟合：

\[
y_{i,\tau}
=
\alpha_{problem(i)}
+
\beta_{\tau}
+
\gamma\,\mathrm{relpos}_{i,t}
+
\delta\,\mathrm{steplen}_{i,t}
+
\epsilon_{i,t}.
\]

报告 \(\tau=-2,-1,0,+1,+2\) 的 clustered bootstrap CI；重点是 \(\tau=0\) 的局部跃迁，不接受只在 \(+2\) 后累积的 retrospective 效应替代。

### 7.2 same-problem paired test

同题内构造 wrong-correct 差：

\[
\Delta_p
=
\operatorname{median}_{i\in wrong(p)}A_i
-
\operatorname{median}_{j\in correct(p)}A_j.
\]

以 problem cluster bootstrap 报告 median \(\Delta_p\)、CI 与 pair-micro AUROC。

### 7.3 增量检验

对 Claim 2 比较：

- nuisance + LID + rewiring；
- nuisance + LID + rewiring + holonomy。

读取器必须 outer-fold 拟合，报告 delta-AUROC 与 CI；增量分数仅是 Claim 2 的辅助证据。

## 8. 资源与运行顺序

### 存储

fp16 whole-layer states：

\[
2N_{step}LD\text{ bytes}.
\]

24k steps、32 layers、4096 hidden 约 6.3 GB。Stage-A 流式写 run-unique fp16 memmap，manifest 最后原子提交；不保留全量 float32 list，也不保存 flattened 副本。Stage-B 默认只写 field 与 provenance，不复制 raw states。

### 计算

- 抽取：1×7B/8B GPU，逐链 teacher forcing；
- geometry：CPU NumPy，默认 JL=64，reference cap=256 chains；
- bootstrap：可并行 CPU；
- synthetic/S0-S2 必须在申请大规模 GPU 前完成。

### 顺序

1. S0-S2；
2. G0 exact smoke；
3. G1-G3 稳定性；
4. 冻结 config；
5. M1/P1；
6. X1/X2；
7. N1-N3；
8. 只在全部 gate 通过后写论文主结果。

## 9. Go / no-go

### 数据门

- exact token/offset/range alignment = 100%；
- whole-layer coverage = 100%；
- OOF LID/connection/holonomy coverage 分别 ≥0.99/0.95/0.90；
- 每 fold reference chains ≥max(\(k_{topology},k_{tangent}\))。

### 几何门

- identical-layer depth rewiring = 0；
- gauge transform 不改变 \(\kappa\)；
- separable synthetic 的 \(\kappa\) 低于 interaction synthetic；
- 64/128-d JL 的主 field rank correlation ≥0.9；
- \(k\)-sweep 主效应方向一致。

### 论文门

- Claim 1 与 Claim 2 至少各有一个预注册主数据集 CI 不跨 0；
- holonomy 增量不能完全由 rank singularity、position 或 length 解释；
- 跨任务至少两项同方向；
- 若 tangent stability 失败，整体退到 neighborhood-lineage holonomy，不再叠加其它 detector；
- 若 Claim 2 失败，只保留 descriptive layer-time atlas，不宣称新机制。
