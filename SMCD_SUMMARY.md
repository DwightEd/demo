# SMCD 工作总结：推理幻觉检测的几何信号

> 截至 2026-06-24。本文档事无巨细地记录已做实验、发现、最终确认指标、可行性铁证与理论依据。
> 代码库：`github.com/DwightEd/demo`（分析）+ `github.com/DwightEd/hallucination-detection`（工程化提取）。

---

## 1. 研究问题与定位

**任务**：ProcessBench 步级推理幻觉检测——给定一条多步数学推理链，定位**第一个出错步**（step-level first-error detection）。
**设定**：Llama-3.1-8B-Instruct，对 ProcessBench 给定解做 **teacher-forcing** 前向，读第 14 层隐状态。这是"审阅他人推理"的设定（隐状态是模型读外来文本时产生的），其真实推理版本见 §9。
**目标**：构建**自己的**几何检测量，在指标上**超越**现有 SOTA 熵基线（EDIS），而非对其增量修改——顶会中稿的前提。

**核心量 κ（kappa）**：一步内所有 token 的单位方向向量的指数加权平均合成长度（mean resultant length）= 方向集中度。错误 ⇒ 方向发散 ⇒ κ 低。

---

## 2. 数据与方法学铁律

**数据规模（全 ProcessBench）**：

| subset | 链(response) | 步(step) | 错误步率 |
|---|---|---|---|
| gsm8k | 395 | 1560 | 13.1% |
| math | 982 | 4339 | 13.5% |
| omnimath | 998 | 4788 | 15.9% |
| olympiadbench | 997 | 5816 | 11.3% |

**四条方法学铁律（贯穿所有实验）**：
1. **GroupKFold by chain**：按链分折，杜绝同链 step 跨 train/test 泄漏。
2. **Bootstrap CI**：对链重采样 200 次给增量的 95% 置信区间——CI 是否跨 0 是"信号真假"的判据。
3. **Bucket AUROC（长度控）**：按 step token 长度分 5 桶加权 AUROC = 诚实的、去长度混淆的数。长度 = 难度的代理，必须控。
4. **Strict 基线增量**：任何几何增量必须在 **[自己的熵 + EDIS + 长度]** 交叉拟合基线之上检验，而非在弱基线(裸 resultant)之上——谱形状就是这样死的（裸基线 +0.038 → strict +0.001）。

## 3. 核心指标与主结果（detector_full 主表）

**OURS** = 多层 κ 家族（层 [2,6,10,14,18,22,26,30] 的 resultant/coherence 堆成特征向量）+ **自研熵 battery**（mean/var/max/range/slope/late），过 **LDA（类内白化）**，**零 EDIS**。

**主表（AUROC，GroupKFold + bootstrap CI）**：

| config | OURS-LDA | EDIS(基线) | OURS−EDIS 增量 [95% CI] |
|---|---|---|---|
| gsm8k | **0.840** | 0.719 | **+0.118 [+0.084, +0.154]** |
| math | **0.797** | 0.717 | **+0.081 [+0.065, +0.098]** |
| omnimath | **0.802** | 0.754 | **+0.048 [+0.035, +0.063]** |
| olympiad | **0.803** | 0.753 | **+0.050 [+0.033, +0.063]** |

**四个 config 的 CI 全部清晰 > 0** —— 显著超越 EDIS。

**Strict 增量**（几何在 [自己的熵 + EDIS + 长度] 之上）：+0.110 / +0.059 / +0.032 / +0.046 —— 即使把 EDIS 也放进基线，几何仍有独立增量。
**Bucket（长度控）AUROC**：OURS 0.795 / 0.757 / 0.750 / 0.755 —— 长度控后依然全面领先，证明不是长度伪迹。
**LDA ≈ logistic**：换分类器结论不变，信号是线性可分的。

**支撑实验**：
- `detector2.py`（单层+自研熵，零 EDIS）：OURS 0.781/0.771/0.787/0.775 vs EDIS 0.719/0.717/0.754/0.753，geom over strict +0.049/+0.035/+0.017/+0.017。
- `kappa_ceiling.py`：全层 κ 家族 logistic 0.805/0.743/0.771/0.757 > 单层 L14 0.772/0.703/0.702/0.703，bucket 同升（长度干净），GBM 更差（信号线性）。
- `layer_key.py`：L14 单层最强 + leave-one-out 掉得最多；信号住在中层 10-22，早层(2/6)/晚层(26/30)弱，无单层独大。
- `complementarity.py`：corr(geom,EDIS)=0.33-0.48，corr(geom,varentropy)≈0，corr(EDIS,varentropy)=0.76-0.78；几何与熵真正正交，cross-ceiling UNION 0.66-0.73 ≫ 各自。

## 4. 理论依据（vMF 方向统计）

**von Mises–Fisher 模型**：把一步内的 token 方向看作单位球面上的样本。
- **合成长度 R̄ = ‖(1/n)Σûᵢ‖ 是 vMF 集中度 κ 的充分统计量**——意味着在"集中度"这一维上，R̄ 已抓全，没有别的标量能加（解释了为什么所有动态/谱变换零增量，见 §5）。
- **R² 恒等式**：R² = (1/n²)ΣΣ⟨ûᵢ,ûⱼ⟩ = 平均两两余弦 = ‖一阶矩‖²。它只读 Gram 矩阵的总和（=总方差 ∑λ=1−R²），对方向分布/簇/顺序失明。
- **Rayleigh 检验是错的**：两类都远离均匀（R̄~0.6 ≫ 1/√n~0.15），判别器是"κ 相对健康基线的程度"，不是"是否非均匀"。
- **vMF 充分统计量是 (μ̂, κ)**：方向 μ̂ + 集中度 κ 两半。**我们一直只用了 κ，把 μ̂（朝向）扔了**——这是 §7 突破的理论根源。

**Fisher LDA = 类内白化**：白化掉"两类都突出的共同方向" + 投到类均值差 Δμ。这是"去掉两者都很突出的方向"的原理化实现，对 2 模态空间（几何⊥熵）是 Bayes 最优。

**样本量效应（已排除的伪迹）**：E[R²] ≈ ρ² + (1−ρ²)/n。token 越多 R 机械地越低，但该效应随集中度 ρ→1 消失。`kappa_debias.py` 用 n_eff 偏差校正后 AUROC 仅 ±0.007 变化、长度相关只降 ~0.06 → **长度混淆是真难度，不是统计伪迹**。

**关键区分**：κ 测的是**表征连贯**（token 方向是否一致），不是**逻辑连贯**（推理是否有效）。长的分类讨论步 = 逻辑连贯但表征发散 → κ 低。这是 κ 在长/难步上失效的根源（见 §6、§7）。

## 5. 信号侧穷尽（负结果清单——证明 κ 是饱和的集中度信号）

所有"动态/相对/形状"变换都在 **strict 基线 + cross-fit + bootstrap CI** 下零增量，逐一证伪：

| 假设 | 脚本 | 结果（增量 [CI]） | 结论 |
|---|---|---|---|
| κ 突降(drop) | `kappa_drop.py` | P3 增量 +0.001/+0.002/+0.003/+0.000，CI **全跨 0** | 突降不独立于 κ 水平 |
| 低 κ 内突降 | `kappa_lowk.py` | +0.020[-0.015,+0.058]/.../+0.011[-0.003,+0.025]，**全跨 0** | 低 κ 内突降也死，43% vs 22% 是 κ 残差 |
| 非单调/U形 | `kappa_shape.py` | cross-fit 杀掉 in-sample bump；错误率随 κ 单调下降，无 U 形 | "高κ自信错误"证伪 |
| 偏差校正 | `kappa_debias.py` | ±0.007，bucket 不变 | 样本量伪迹是小头 |
| 谱形状 | `dir_spectrum.py` | strict 上 +0.001（裸基线 +0.038） | 结构 vs 各向同性发散不可分 |
| 顺序/跨步 | `cross_step_coherence.py` | 0.51-0.59 弱，+0.002 零增量（但 corr 0.11 真正交） | 步内几何饱和在一个标量 |

**定论**：步级几何错误信号 = **绝对多层 κ 水平**，vMF 集中度充分统计量；集中度这一维已饱和。（注：此"饱和"只针对集中度半边，朝向半边是活的——§7 纠正。）

## 6. 粒度分工发现（几何=局部 / 熵=全局）

把任务从步级换到**响应级**（整条链对错），信号天平翻转：

**`resp_detect.py`（响应级 AUROC）**：
- GDIS（EDIS 式套 κ 轨迹）：0.555-0.619 —— **弱**，κ 的动态聚合无用；
- min-κ：gsm8k 0.740 / math 0.698 / omnimath 0.658 / olympiad 0.640；
- **EDIS(熵)：0.736 / 0.731 / 0.812 / 0.739** —— 响应级、尤其难任务，熵动态碾压几何。

**`resp_cusum.py`（分段无关在线 CUSUM，长度控 bkt）**：
- gsm8k：CUSUM 0.739 > min-κ 0.723 > mean 0.718；
- omnimath：CUSUM 0.591 ≈ mean 0.600 > min-κ 0.579（min-κ raw 0.636 是长度虚高，长度控后塌到 0.579；CUSUM 长度鲁棒）。

**`kappa_bylen.py`（κ AUROC 按步长三分位）**：
- gsm8k 0.801→0.702→0.707；math 0.697→0.639→**0.586**；omnimath 0.683→0.572→**0.568**；olympiad 0.585→0.561→0.618。
- **κ 在长步崩，EDIS 在长步稳/升** → 长复合步（分类讨论）几何失效；
- `kappa_corr ≈ kappa` 每层成立 → 样本量伪迹 #3 **定论排除**，长步崩是内容本身发散（#2），非统计。

**跨模态对照（美的结论）**：对**熵**，信号在轨迹形状（EDIS > mean-entropy）；对**几何**，信号在静态水平（min-κ > GDIS）。两模态充分统计量不同。
**粒度分工**：几何=局部（步级首错定位，OURS 0.80-0.84）；熵动态=全局（响应级可靠性，EDIS 0.73-0.81）。跨粒度互补。

## 7. ★ 朝向突破：μ̂ 是难任务信号（probe_ceiling.py）

**动机**：竞品 2601 的训练探针吃完整隐状态而我们只用 κ 标量；vMF 充分统计量 (μ̂, κ) 的朝向半边 μ̂ 一直没用。
**做法**：用与竞品相同的 step_exp 池化（wₙ=(n-1)/(L-1)）把逐 token respcloud（JL-256）按步池化 → 256 维步向量，leak-free 交叉拟合线性探针，长度控 bucket，分解为 朝向 μ̂ / 集中度 κ / 幅度。

**结果（bucket(len) AUROC，L14）**：

| | omnimath（难） | gsm8k（易） |
|---|---|---|
| κ 标量（集中度） | 0.565 | 0.708 |
| **朝向 μ̂（256维方向）** | **0.763** | 0.715 |
| 幅度（pooled-norm） | 0.563 | 0.701 |
| 完整表征探针（256维） | 0.766 | 0.734 |
| **μ̂ + κ（完整 vMF）** | **0.771** | 0.715 |

**三重结论（一次解决三个问题）**：
1. **难任务上增益纯粹来自朝向 μ̂**（0.763 ≈ 完整探针 0.766 ≫ κ 0.565；幅度 0.563 无用）；易任务上 κ≈μ̂。
2. **"为什么难任务对 κ 难"解决**：κ 只看集中度、扔了朝向，而难任务的错是**"方向集中但指错了"（连贯但偏目标）**，住在被扔的那半。
3. **竞品 2601 高分的"真方法"部分 = 隐式抓了 μ̂**；我们显式化 + vMF 理论化。
4. **μ̂+κ（完整充分统计量）最高 0.771，已超 detector_full 的 OURS bkt 0.750**。

> 纠正：早先"几何饱和在 κ"是错的——只饱和了集中度半边；**朝向 μ̂ 是一直没动的活信号**。

**下一步原理化**：重抽 qvec（目标方向）后，用 cos(μ̂, q) = **目标对齐**替黑箱探针 → 从"判别方向"升级到可解释的"是否偏离目标"单轴。

## 9. 真实推理验证路径（own-generation）

ProcessBench 主结果是 teacher-forcing 读**外来解**（离分布）。真实推理版已就绪：
- `10_sample_and_extract.py`：模型**自采样生成** K 条解 → **终答对错**自动判（零步级标签）→ 同题 K 采样**难度天然受控**。
- `extract_features.py --source sampled`：同提取器在自己的生成上抽，带 `is_correct`。
- **为什么 teacher-forcing 自己的生成 = 真实生成态**：位置 t 的隐状态只由前缀决定，喂回自己生成的序列复现逐位置态。
- **决定性测试**：`resp_cusum` 在 sampled 数据上 bkt-AUROC > 0.7 ⇒ 几何是真实推理内在属性、部署主张成立；~0.5 ⇒ 信号是"读外来文本"特有，需重定位。
- **现状**：链路就绪，待跑（需补 `--store_clouds` 重抽采样源）。

## 11. 最有力证据汇总（"指标可行"的铁证）

1. **主表显著性**：OURS 超 EDIS 四 config CI 全清晰 > 0（+0.048~+0.118），且 **strict 增量**（在 [熵+EDIS+长度] 之上）仍 +0.032~+0.110 —— 不是把别人的信号换皮。
2. **长度干净**：bucket(长度控) AUROC OURS 0.750-0.795 仍领先 —— 排除长度=难度伪迹。
3. **偏差校正不变**：`kappa_debias` ±0.007，kappa_corr≈kappa —— 排除样本量统计伪迹。
4. **穷尽负结果**：jump/shape/drop/lowk/谱/跨步全部增量 CI 跨 0 —— 信号定位在"绝对 κ 水平"无歧义。
5. **正交性**：corr(几何,熵)≈0，UNION ≫ 各自 —— 几何是独立传感器。
6. **★ 朝向铁证**：难任务 μ̂ 0.763 ≫ κ 0.565、幅度 0.563 无用、μ̂+κ 0.771 最高（长度控）—— 完整 vMF 充分统计量是对的，且解释了难任务退化。
7. **理论闭环**：R̄ = vMF 集中度充分统计量（证饱和）；(μ̂,κ) = 完整统计量（证朝向是活信号）；Fisher = 类内白化。
