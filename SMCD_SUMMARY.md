# SMCD 工作总结：推理幻觉检测的几何信号

> 截至 2026-06-24。本文档事无巨细地记录已做实验、发现、最终确认指标、可行性铁证与理论依据。

## 1. 研究问题与定位

**任务**：ProcessBench 步级推理幻觉检测——给定一条多步数学推理链，定位**第一个出错步**（step-level first-error detection）。
**设定**：Llama-3.1-8B-Instruct，对 ProcessBench 给定解做 **teacher-forcing** 前向，读第 14 层隐状态。这是"审阅他人推理"的设定（隐状态是模型读外来文本时产生的），其真实推理版本见 §9。
**目标**：构建**自己的**几何检测量，在指标上**超越**现有 SOTA 熵基线（EDIS），而非对其增量修改——顶会中稿的前提。

**核心量 κ（kappa）**：一步内所有 token 的单位方向向量的指数加权平均合成长度（mean resultant length）= 方向集中度。错误 ⇒ 方向发散 ⇒ κ 低。

## 2. 数据与方法学铁律

**数据规模（全 ProcessBench）**：

| subset | 链(response) | 步(step) | 错误步率 |
|---|---|---|---|
| gsm8k | 395 | 1560 | 13.1% |
| math | 982 | 4339 | 13.5% |
| omnimath | 998 | 4788 | 15.9% |
| olympiadbench | 997 | 5816 | 11.3% |

**四条方法学铁律（贯穿所有实验）**：
1. **GroupKFold by chain**：按链分折，杜绝同链 step 跨 train/test 泄漏。
2. **Bootstrap CI**：对链重采样 200 次给增量的 95% 置信区间——CI 是否跨 0 是"信号真假"的判据。
3. **Bucket AUROC（长度控）**：按 step token 长度分 5 桶加权 AUROC = 诚实的、去长度混淆的数。长度 = 难度的代理，必须控。
4. **Strict 基线增量**：任何几何增量必须在 **[自己的熵 + EDIS + 长度]** 交叉拟合基线之上检验，而非在弱基线(裸 resultant)之上——谱形状就是这样死的（裸基线 +0.038 → strict +0.001）。

**OURS** = 多层 κ 家族（层 [2,6,10,14,18,22,26,30] 的 resultant/coherence 堆成特征向量）+ **自研熵 battery**（mean/var/max/range/slope/late），过 **LDA（类内白化）**，**零 EDIS**。

**主表（AUROC，GroupKFold + bootstrap CI）**：

| config | OURS-LDA | EDIS(基线) | OURS−EDIS 增量 [95% CI] |
|---|---|---|---|
| gsm8k | **0.840** | 0.719 | **+0.118 [+0.084, +0.154]** |
| math | **0.797** | 0.717 | **+0.081 [+0.065, +0.098]** |

**⚠ 比较公平性（重要，别被 0.84 vs 0.72 误导）**：OURS 是**多层多特征 + LDA 分类器**，EDIS 是**单个手工分数**。直接比 0.84 vs 0.72 不公平，那个 0.12 落差有一部分只是"多特征 + 分类器"带来的。要看公平的数，有两个：
- **单标量对单标量**：κ 单标量 0.70–0.77 ≈ EDIS 单分数 0.72–0.75，**基本打平**；几何并不是单点上碾压熵。
- **同分类器内的几何增量（这才是主结果该讲的）**：把几何放进一个**已经含 EDIS** 的分类器，它额外带来 **+0.110 / +0.059 / +0.032 / +0.046**（下面 strict 增量）。这是"几何比熵多了多少"的诚实数。

**Strict 增量**（几何在 [自己的熵 + EDIS + 长度] 之上）：+0.110 / +0.059 / +0.032 / +0.046 —— 即使把 EDIS 也放进基线，几何仍有独立增量。**这条 + CI 是论文该当头条的公平结果，不是 0.84 vs 0.72。**
**Bucket（长度控）AUROC**：OURS 0.795 / 0.757 / 0.750 / 0.755 —— 长度控后依然全面领先，证明不是长度伪迹。
**LDA ≈ logistic**：换分类器结论不变，信号是线性可分的。

**支撑实验**：
- `detector2.py`（单层+自研熵，零 EDIS）：OURS 0.781/0.771/0.787/0.775 vs EDIS 0.719/0.717/0.754/0.753，geom over strict +0.049/+0.035/+0.017/+0.017。

### 4.1 指标精确计算

一步 j、层 ℓ，该步 token 位置 t=0..Lⱼ−1，隐状态 h_t^(ℓ)：

```
û_t = h_t^(ℓ) / ‖h_t^(ℓ)‖                          # 单位方向
w_t = exp(t / (Lⱼ−1))                              # step_exp 指数权（偏后段 token）
κ_j = ‖Σ_t w_t û_t‖ / (Σ_t w_t)  ∈ [0,1]          # 指数加权合成长度 = 集中度
```

**κ 家族（4 个变体）**：`resultant`（上式）、`resultant_unif`（均匀权）、`resultant_bulk`（零掉 massive 维后的合成长度）、`coherence`。
**OURS 检测量**：`column_stack( [κ家族 × 层 2,6,10,14,18,22,26,30] + [熵 battery] ) → LDA`；熵 battery = 逐 token 熵的 `{mean, var, max, range, slope, late}`。零 EDIS。

### 4.2 von Mises–Fisher 与 R² 恒等式

把一步内 token 方向看作单位球面样本。
- **合成长度 R̄ = ‖(1/n)Σûᵢ‖ 是 vMF 集中度 κ 的充分统计量** → 集中度这一维 R̄ 已抓全，无别的标量能加（解释所有动态/谱变换零增量，§5）。
- **R² 恒等式**（均匀权，说明本质）：
  ```
  R̄² = ‖(1/n)Σûᵢ‖² = (1/n²)(Σûᵢ)·(Σûⱼ) = (1/n²)ΣᵢΣⱼ⟨ûᵢ,ûⱼ⟩ = 平均两两余弦 cos θᵢⱼ
  ```
  即 **κ² = 全体 token 方向的平均两两对齐度**；1−R̄ = 球面方差。κ 只读"总对齐"，对**指向哪(μ̂)**和簇结构**失明** → §7 朝向是另一半。
- **vMF 充分统计量 = (μ̂, κ)** 两半：方向 μ̂ + 集中度 κ。**我们一直只用 κ、扔了 μ̂** = §7 突破根源。
- **Rayleigh 检验是错的**：两类都远离均匀（R̄~0.6 ≫ 1/√n），判别器是"κ 相对健康基线的程度"。

### 4.3 vMF 验证现状（诚实）

vMF 是**模型**，其核心可检验预言"充分统计量=(μ̂,κ)、别的都不加判别力"被**间接但强力证实**（§5 全部负结果 + kappa_test「Bayes-LR=monotone(−κ)」+ §7 唯一能加的是 μ̂）。**但未做直接分布拟合优度检验**（KS/Anderson-Darling 判 token 方向是否真服从单一 vMF vs 混合/重尾）——这是论文待补的 pilot。

### 4.4 Fisher、样本量、表征 vs 逻辑

- **Fisher LDA = 类内白化**：白化掉"两类共同突出方向" + 投到 Δμ；对 2 模态空间（几何⊥熵）Bayes 最优。
- **样本量效应（已排除）**：E[R²] ≈ ρ² + (1−ρ²)/n，随 ρ→1 消失。`kappa_debias` 偏差校正后 AUROC ±0.007、长度相关只降 ~0.06 → 长度混淆是真难度非统计伪迹。
- **关键区分**：κ = **表征连贯**（方向一致）≠ **逻辑连贯**（推理有效）。长分类讨论步 = 逻辑连贯但表征发散 → κ 低（§6、§7 退化根源）。

## 5. 单调性（可挖掘性正证据）+ 动态分析全表

### 5.1 单调性 = κ 是可挖掘方向的正证据（kappa_test / kappa_shape）

**κ 均值，正确步 vs 错误步**（错误步 κ 系统性更低）：

| config | κ̄(correct) | κ̄(error) | raw monotone(−κ) | raw Bayes-LR |
|---|---|---|---|---|
| gsm8k | 0.634 | 0.593 | 0.772 | 0.773 |
| math | 0.595 | 0.558 | 0.703 | 0.703 |
| omnimath | 0.578 | 0.543 | 0.702 | 0.702 |
| olympiad | 0.579 | 0.547 | 0.703 | 0.703 |

**raw 下 Bayes 最优 1-D 似然比 = monotone(−κ)**（四 config 全相等）→ **κ 水平本身就是 Bayes 最优 1-D 分数**，没有非线性可挖。

**中等长度步、错误率按 κ 十分位（低κ→高κ）**——清晰单调下降，无 U 形：

|---|---|---|---|---|---|---|---|---|---|---|

| math | 0.26 | 0.18 | 0.15 | 0.08 | 0.14 | 0.09 | 0.08 | 0.11 | 0.07 | 0.05 |
| omnimath | 0.21 | 0.16 | 0.16 | 0.14 | 0.15 | 0.10 | 0.14 | 0.11 | 0.11 | 0.09 |

**交叉拟合杀掉过拟合 bump**（kappa_shape 长度控，xfit Bayes-LR ≤ monotone）：gsm8k 0.656≤0.707、math 0.576≤0.610、omnimath 0.560≤0.571、olympiad 0.577≈0.578。

**★ 二阶矩谱在难任务上真加（scatter_spectrum，干净正结果）**：单位 token 向量的散布矩阵特征谱（Bingham/Watson 充分统计量），在难任务上带 κ 漏掉的发散结构信息：
- omnimath：eff_rank 单独 bkt 0.610 > κ 0.565（无监督单标量、长度控就超 κ）；top-5 谱向量 over [κ+len] **+0.021 [+0.013,+0.031]，CI 清晰 > 0**；
- gsm8k：谱 over [κ+len] −0.008 [−0.013,−0.005]，易任务 κ 已够。
- 机制：κ 把"各向同性扩散（真错误）"和"沿低维子空间结构化铺开（分类讨论）"都看成低 κ；eff_rank/谱能分开。**这是干净的——无监督、长度控、GroupKFold、有 CI、低维可解释（5 个特征值），不是 §7 那种监督探针。**
- 待办：math/olympiad（需 respcloud 重抽）确认是否是普遍的难任务性质；把谱/eff_rank 加进 detector_full 看难 config 的提升。

**关于朝向（§7）**：无监督方向偏离标量（3b）是空的；监督 256 维探针的 0.76 是 2601 路线、非干净几何信号、非差异化；可解释单标量朝向 cos(步方向, qvec) 未测，是开放项。

## 6. 粒度分工发现（几何=局部 / 熵=全局）

把任务从步级换到**响应级**（整条链对错），信号天平翻转：

**`resp_detect.py`（响应级 AUROC）**：
- GDIS（EDIS 式套 κ 轨迹）：0.555-0.619 —— **弱**，κ 的动态聚合无用；
- min-κ：gsm8k 0.740 / math 0.698 / omnimath 0.658 / olympiad 0.640；
- **EDIS(熵)：0.736 / 0.731 / 0.812 / 0.739** —— 响应级、尤其难任务，熵动态碾压几何。

**`resp_cusum.py`（分段无关在线 CUSUM，长度控 bkt）**：
- gsm8k：CUSUM 0.739 > min-κ 0.723 > mean 0.718；
- omnimath：CUSUM 0.591 ≈ mean 0.600 > min-κ 0.579（min-κ raw 0.636 是长度虚高，长度控后塌到 0.579；CUSUM 长度鲁棒）。

**`kappa_bylen.py`（κ AUROC 按步长三分位）**：
- gsm8k 0.801→0.702→0.707；math 0.697→0.639→**0.586**；omnimath 0.683→0.572→**0.568**；olympiad 0.585→0.561→0.618。
- **κ 在长步崩，EDIS 在长步稳/升** → 长复合步（分类讨论）几何失效；
- `kappa_corr ≈ kappa` 每层成立 → 样本量伪迹 #3 **定论排除**，长步崩是内容本身发散（#2），非统计。

**跨模态对照（美的结论）**：对**熵**，信号在轨迹形状（EDIS > mean-entropy）；对**几何**，信号在静态水平（min-κ > GDIS）。两模态充分统计量不同。
**粒度分工**：几何=局部（步级首错定位，OURS 0.80-0.84）；熵动态=全局（响应级可靠性，EDIS 0.73-0.81）。跨粒度互补。

## 7. 朝向问题：监督探针 vs 无监督标量（归因纠正，不是突破）

**早先这一节被标成"朝向 μ̂ 突破"，是错误归因。纠正如下。**

`probe_ceiling.py` 跑的是：把逐 token 隐状态按步 exp 池化成 256 维向量，喂一个**有监督** logistic 探针。它分三路——`MU`（单位方向）、`KA`（集中度标量）、`V`（原始向量）。结果（bucket(len)，L14）：

| | omnimath（难） | gsm8k（易） |
|---|---|---|
| κ 标量（集中度，无监督单标量） | 0.565 | 0.708 |
| 监督探针·单位方向 MU（256维） | 0.806→bkt 0.763 | 0.715 |
| 监督探针·原始向量 V（256维） | 0.795→bkt 0.766 | 0.734 |
| 幅度标量 | 0.563 | 0.701 |

**为什么这不是"vMF 朝向半边"**：
1. `MU(256维)` 0.763 ≈ `V(256维)` 0.766。两者几乎相等，说明这个探针量的是**整个 256 维表征在监督下能分多少**，不是 vMF 意义上的方向 μ。
2. 真正测"vMF 朝向半边"应该用**无监督单标量** cos(步方向, 参照)，那就是 3b 的方向偏离量——**它是空的（增量 −0.001/+0.004）**。
3. 所以诚实的结论是：**vMF 的朝向半边（正确测法）没有信号**；那个 0.76 只是"**监督探针吃完整池化表征**"，是 trivial 的（监督当然赢无监督标量）。
4. **它和 §8 矛盾**：我在 §8 批评竞品 2601"重监督、吃完整隐状态"，却把一个同样重监督、吃完整表征的探针当成自己的突破。不能这样。这个 0.76 是 2601 / TaT 路线，不是我们的差异化。

**唯一站得住的中性事实**：难任务上，监督探针（0.76）确实比无监督 κ（0.57）多拿不少 → **κ 这一套可解释手工特征在难任务上是不完整的**。但补全它要么靠更多几何特征（已证伪，§5 全空），要么退化成"在完整表征上训探针"（=变成 2601）。这是对我们叙事的**挑战**，不是突破。

**真正未解的开放问题**：一个**可解释的单标量朝向量**（如 cos(步方向, 目标向量 qvec)）到底有没有信号——没测过（qvec 没抽）。这才是"朝向是不是活信号"的正确实验，待全维重抽后做。

**我们独有的、可辩护的核心**：① 机制/理论（κ=vMF 集中度充分统计量、μ̂=朝向半边、几何⊥熵、粒度分工）；② 方法论严谨（难度/长度混淆 + answer-aware 标签泄漏的揭露 + strict 增量 + CI）。**不是**"首个流式"、**不靠**单纯刷高 AUROC。

## 9. 真实推理验证路径（own-generation）

ProcessBench 主结果是 teacher-forcing 读**外来解**（离分布）。真实推理版已就绪：

## 11. 最有力证据汇总（"指标可行"的铁证）

1. **公平的主结果**：几何放进**含 EDIS 的同一分类器**仍 +0.032~+0.110，CI 全 > 0（strict 增量）—— 这是"几何比熵多带了多少"的诚实数（不是 0.84 vs 0.72 那个不公平头条）。
2. **长度干净**：bucket(长度控) AUROC OURS 0.750-0.795 仍领先 —— 排除长度=难度伪迹。
3. **偏差校正不变**：`kappa_debias` ±0.007，kappa_corr≈kappa —— 排除样本量统计伪迹。
4. **穷尽负结果**：jump/shape/drop/lowk/谱/跨步/无监督朝向偏离 全部增量 CI 跨 0 —— 无监督几何信号定位在"绝对 κ 水平"无歧义。
5. **正交性**：corr(几何,熵)≈0，UNION ≫ 各自 —— 几何与熵是独立传感器。
6. **单调性**：错误率随 κ 十分位单调下降（gsm8k 0.22→0.04），raw Bayes-LR = monotone(−κ) —— κ 是可挖掘方向。
7. **理论**：R̄ = vMF 集中度充分统计量（证集中度半边饱和）；Fisher = 类内白化。

> ❌ 已撤回的"证据"：早先把"256 维监督探针 0.763"当成"朝向 μ̂ 是活信号 / vMF 突破"——错误归因（§7）。监督探针吃完整表征 ≈ 2601 路线，不是干净几何信号；正确测法（无监督朝向标量）是空的。可解释单标量朝向尚未测。
