# 方法重设计报告：整层 Layer-Time 表征几何

生成时间：2026-07-13 02:13（Asia/Shanghai）
运行 ID：demo-method-redesign-20260713
阶段：idea-discovery revision after user steer

## 1. 结论先行

用户的收束是正确的：当前项目不应继续把 anchor、uncertainty、Gram、hazard、HMM 和 intervention 串成一个大系统。那会把多个尚未独立成立的假设缝在一起，也不能回答“整层表征几何如何演化”。

新的唯一主线是：

> 把每条推理链表示成网络深度 \(\ell\) 与推理时间 \(t\) 上的局部几何场；用同一个 phase-matched kNN chart 估计 LID、邻域谱系和 tangent fiber，再用 layer/time 两类局部 connection 的 plaquette holonomy 检验二维路径相容性。

描述性名称：Phase-Matched Layer-Time Tangent-Bundle Geometry。名称不是论文贡献；贡献是否成立由真实实验决定。

## 2. 旧方案为什么不适合

### 2.1 层轴被过早消失

现有 representation_geometry 把：

\[
Z_{i,t}\in\mathbb R^{L\times D}
\]

先压成：

\[
\operatorname{vec}(Z_{i,t})\in\mathbb R^{LD},
\]

再计算一个 LID 和一个平均跨层不稳定度。这不是“逐层几何演化”，只能得到层混合后的状态分数。

spectral_chain_dynamics 同样把完整 \(T\times L\times D\) 变为 \(T\times(LD)\)，因此深度轴无法恢复。

### 2.2 旧跨层 topology 有人为噪声

旧代码每层各抽一个独立 Gaussian projection，再比较 kNN identity。即使两层输入完全相同，也能得到很高的邻域不稳定度。这是投影变化，不是 representation rewiring。

新实现所有层共享同一个 JL projection；identical-layer 回归测试要求 depth rewiring 精确为 0。

### 2.3 旧切分会泄漏内部同题多样本

旧 geometry/spectral 按 chain ID 分 fold；同一道题的不同 sampled response 会落入 train/test 两侧。新主线从 reference 构造开始就按 problem_id 切分。

### 2.4 旧几何受标签塑形

旧 representation 使用 healthy/first-error reference，旧 spectral 还按 label 平衡 landmark。它们可以作为监督诊断 baseline，但不能支持 label-free manifold evolution。

新 field 构造完全不读取 correctness 或 first-error label；测试中任意翻转标签，完整 field 不变。

### 2.5 时间 reference 不能混合早中晚状态

若所有训练 step 混成一个 reference cloud，所谓 \(X_{\ell,t}\) 实际不是 cell-wise geometry。新实现把每条训练链插值到查询链相同的 normalized phase；每条参考链在每个 cell 只贡献一个状态，参考身份跨层与跨时间保持一致。

## 3. 统一方法

### 3.1 显式 state tensor

\[
z_{i,t,\ell}
=
\frac1{|S_{i,t}|}\sum_{p\in S_{i,t}}h_{i,p}^{(\ell)},
\qquad
Z_i\in\mathbb R^{T_i\times L\times D}.
\]

主 schema 不再 flatten。正式结果要求 raw hidden state、mean pooling、连续层和 exact token span。

### 3.2 phase-matched reference-chain cloud

\[
X_{\rho,\ell}
=
\{\widetilde z_{j,\rho,\ell}:j\in\mathcal R_{\mathrm{train}}\}.
\]

训练 reference 按 problem/sample 平衡，标签不参与。

### 3.3 同一个局部 chart

kNN 半径给 LID：

\[
\widehat d
=
\left[
\frac1{k-1}\sum_{m=1}^{k-1}\log\frac{r_k}{r_m}
\right]^{-1}.
\]

kNN 身份给 depth/time neighborhood lineage：

\[
\Omega
=
1-\frac{|\mathcal N_a\cap\mathcal N_b|}{k}.
\]

同一邻域 SVD 给 tangent basis，LID 决定 fiber rank：

\[
q=\min(q_{\max},\operatorname{round}(\widehat d),\operatorname{rank}_{num}).
\]

所以 LID、topology、tangent 不是三个插件，而是同一个局部 chart 的半径、身份与方向结构。

### 3.4 局部 connection 与 holonomy

相邻 cell 用相同参考链 ID 的 paired tangent coordinates 拟合正交 transport。每个 layer-time 方格有两条路径：

\[
P_{\ell t}=R^tR^\ell,
\qquad
P_{t\ell}=R^\ell R^t.
\]

核心量：

\[
\kappa
=
\frac{\|P_{\ell t}-P_{t\ell}\|_F}{2\sqrt q}.
\]

四角 rank 不一致时额外输出 rank_singularity，而不是偷偷强制固定维数。

## 4. 创新边界

已有工作已经覆盖很多单轴或标量版本，因此不能把“逐层 LID”“kNN overlap”或“二维 heatmap”单独宣称为创新：

- Geometry of Hidden Representations 已研究 layer-only kNN neighborhood rearrangement；
- Geometry of Decision Making 已画全层 ID 曲线；
- Reasoning Emerges from Constrained Inference Manifolds 已保存完整 \(h_{\ell,t}\)，但主要将每层时间轨迹压成标量；
- The Spectral Geometry of Thought 已构造 layer-time 标量场；
- Truth as a Trajectory 与 LLM Reasoning as Trajectories 研究一维轨迹输运。

当前最可防守的空隙是：

> label-free、phase-matched、problem-grouped 的局部 layer-time tangent connection，以及两轴 transport 的路径非交换性。

项目内部已有 hcr_holo_audit，因此也不能笼统写“首次把 holonomy 用于 reasoning”。必须强调对象与估计协议不同：local tangent bundle、phase-matched reference identities、LID-adaptive rank、layer-time plaquette。

1. metrics.py 新增 [step,layer,hidden] mean pooling，旧 flat 仅兼容视图。
2. extraction.py 新增 geometry-only；all 表示全部 post-block depths。

5. layer_time_geometry.py 实现 problem-group OOF、phase interpolation、shared JL、LID、neighbor lineage、rank-aware local connection、holonomy、coverage hard gate。

7. canonical adapter 可读取 exact sv_vec_mean；reasoning-subspace projection 会被 representation guard 拒绝。

## 7. 当前方案仍有的问题

1. normalized-step phase 可能不等价于认知进度；需 token progress 与 arclength 对照。
2. mean pooling 会抹掉 step 内 token-cloud geometry；它是当前统一对象的低分辨率版本。
3. LID MLE 对 reference 数、距离集中和 k 敏感。
4. local tangent 在样本不足或 rank change 时可能不稳定。
5. holonomy 可能只是位置、链长或 rank singularity 的派生效应。
6. phase interpolation 是欧氏线性插值；需要 nearest-phase 与 spline/arc-length robustness。
7. 当前标量 reductions 仍可能引入 best-of-many；必须预注册 layer band/field functional。
8. 真实 artifact 的存储与 I/O 可能成为主要瓶颈。

## 8. 结果分析

历史真实结果已经说明“再加一个 scalar”风险很高：

- spread/resultant 在跨题上较高，但同题显著下降；
- geometry+uncertainty 有过增量，但 scalar HSMM、path kernel、direct Gram/second moment 与 token-stream boundary-free 多数为负或近零；
- qvec anchor 没有稳定增量；
- constraint anchor 结果被位置混杂主导。

这些结果不直接否定 layer-time tangent geometry，但强制新方法证明：

- 同题增量；
- first-error 局部化而非链尾效应；
- 对位置/长度的独立性；
- 几何不变量和负控；
- 跨任务/模型稳定性。

## 10. 优化建议

### 工程

- 对超大数据可在当前 memmap writer 上再增加多盘 shard；
- phase reference 与 projected cloud 按 fold/cache 复用；
- kNN 用 FAISS 或分块矩阵乘法替换大规模全距离；
- fold 并行，bootstrap 按 problem 并行；
- 将 finite coverage、rank coverage、transport residual 写入 summary hard gate。

### 方法

- 以 token-progress/arclength phase 做主稳健性；
- 对 LID 使用 k-sweep median 或 bootstrap posterior，而非单一 k；
- 对 rank singularity 单独建 event map，不混入 holonomy；
- 报告完整 layer-time atlas、peak layer、band width 与 propagation direction；
- scalar AUROC 仅作附属 readout。

### 写作

- 不声称“首次 holonomy”；
- 不把 synthetic wiring test 写成 empirical support；
- 不把稀疏层/step-exp legacy 数据称为 whole-layer mean-state；
- 主张最多两个：局部几何分叉与 layer-time 非交换性。
